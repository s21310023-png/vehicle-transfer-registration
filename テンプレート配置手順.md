# 📄 テンプレートPDF配置手順

## 手順

1. **申請用紙PDFファイル（000228524.pdf）を見つける**
   - ダウンロードフォルダ
   - デスクトップ
   - その他の保存場所

2. **ファイルをコピー**
   - `000228524.pdf` を選択してコピー（Ctrl+C）

3. **templatesフォルダに貼り付け**
   - プロジェクトフォルダ内の `templates` フォルダを開く
   - ファイルを貼り付け（Ctrl+V）

4. **ファイル名を変更**
   - `000228524.pdf` → `transfer_template.pdf` にリネーム

## または、PowerShellで実行

申請用紙PDFがダウンロードフォルダにある場合：

```powershell
Copy-Item "$env:USERPROFILE\Downloads\000228524.pdf" "templates\transfer_template.pdf"
```

## 確認

配置後、以下で確認：

```powershell
Test-Path templates\transfer_template.pdf
```

`True`と表示されればOKです！
