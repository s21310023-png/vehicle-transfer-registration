# 🚀 次のステップ

## 1. 新しいフォルダに移動

PowerShellで新しいフォルダに移動してください：

```powershell
cd "$env:USERPROFILE\Desktop\移転登録③"
```

## 2. 依存関係のインストール

新しいフォルダで依存関係をインストール：

```powershell
npm install
```

これには数分かかる場合があります。

## 3. 環境変数の確認

`.env`ファイルが正しくコピーされているか確認：

```powershell
Test-Path .env
```

`True`と表示されればOKです。

## 4. テンプレートPDFの確認

```powershell
Test-Path templates\transfer_template.pdf
```

`True`と表示されればOKです。

## 5. サーバーの起動

### 方法A: バッチファイル（簡単）
- `start.bat`をダブルクリック

### 方法B: PowerShellで実行
```powershell
npm run dev:all
```

### 方法C: 個別に起動
**ターミナル1（バックエンド）:**
```powershell
npm run server
```

**ターミナル2（フロントエンド）:**
```powershell
npm run web
```

## 6. ブラウザで確認

- http://localhost:5173 にアクセス
- エラーメッセージが消えていれば成功です！

## トラブルシューティング

### エラー: "Cannot find module"
→ 依存関係を再インストール：
```powershell
npm install
```

### エラー: "Port 3000 is already in use"
→ 既にサーバーが起動している可能性があります。別のターミナルで確認してください。

### エラー: "OPENAI_API_KEY is not set"
→ `.env`ファイルを確認し、APIキーが設定されているか確認してください。
