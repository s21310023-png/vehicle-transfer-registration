# ✅ 設定チェックリスト

アプリを起動する前に、以下を確認・設定してください：

## 1. 環境変数の設定（必須）

### 手順：
1. プロジェクトフォルダで `env.example` を `.env` にコピー
2. `.env` ファイルを開く
3. `OPENAI_API_KEY` にあなたのOpenAI APIキーを設定

```env
OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxx
```

**重要**: `.env`ファイルは作成されていません。必ず作成してください！

---

## 2. テンプレートPDFの準備（必須）

### 手順：
1. `templates` フォルダを開く
2. 移転登録申請書のテンプレートPDFを `transfer_template.pdf` という名前で配置

**重要**: `templates/transfer_template.pdf` が存在しません。必ず配置してください！

---

## 3. サーバーの起動

### 簡単な方法（推奨）：
**`start.bat` をダブルクリック**

または、PowerShellで：
```powershell
npm run dev:all
```

### 個別に起動する場合：

**バックエンドサーバー:**
- `start-server.bat` をダブルクリック
- または: `npm run server`

**フロントエンドサーバー:**
- `start-web.bat` をダブルクリック  
- または: `npm run web`

---

## 4. 確認事項

起動後、以下を確認：

✅ ターミナルに「🚀 APIサーバーが起動しました: http://localhost:3000」と表示される
✅ ブラウザで http://localhost:5173 にアクセスできる
✅ エラーメッセージが消えている
