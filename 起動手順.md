# 🚀 Webアプリ起動手順

## ステップ1: 環境変数の設定

`.env`ファイルを作成（または確認）してください：

1. `env.example`を`.env`にコピー
2. `.env`ファイルを開いて、以下を設定：
   ```
   OPENAI_API_KEY=your-openai-api-key-here
   ```

## ステップ2: テンプレートPDFの準備

`templates/`フォルダに以下を配置：
- `transfer_template.pdf` （移転登録申請書のテンプレート）

## ステップ3: サーバーの起動

### 方法A: 両方のサーバーを同時に起動（推奨）

**新しいターミナル（PowerShell）を開いて：**

```powershell
cd C:\Users\s2131\Pictures\Screenshots
npm run dev:all
```

これで、バックエンド（ポート3000）とフロントエンド（ポート5173）が同時に起動します。

### 方法B: 個別に起動

**ターミナル1（バックエンド）:**
```powershell
cd C:\Users\s2131\Pictures\Screenshots
npm run server
```

**ターミナル2（フロントエンド）:**
```powershell
cd C:\Users\s2131\Pictures\Screenshots
npm run web
```

## ステップ4: ブラウザで確認

1. ブラウザで http://localhost:5173 を開く
2. ターミナルに「🚀 APIサーバーが起動しました: http://localhost:3000」と表示されていればOK

## トラブルシューティング

### エラー: "Cannot find module"
→ 依存関係を再インストール：
```powershell
npm install
```

### エラー: "Port 3000 is already in use"
→ ポート3000を使用しているプロセスを終了するか、`server.ts`の`PORT`を変更

### エラー: "OPENAI_API_KEY is not set"
→ `.env`ファイルにAPIキーを設定してください
